Compte le nombre d'enregistrements d'une table, avec une clause WHERE optionnelle.

countTableRows(string $table[, string $where = '' ]) : int

Le nom de la table est d'abord validé pour garantir qu'il ne contient que des caractères autorisés. La requête SQL construite compte les enregistrements correspondant à la clause conditionnelle fournie, ou compte tous les enregistrements si aucune clause n'est spécifiée.

Parameters
$table : string

Nom de la table (doit contenir uniquement des caractères alphanumériques et underscores).

$where : string = ''

Clause SQL optionnelle pour filtrer les enregistrements.

Return values
int —

Nombre d'enregistrements répondant aux critères.

fetchAll()

db.php : 107

Récupère l'ensemble des enregistrements d'une table en appliquant des filtres optionnels.

fetchAll(string $table[, string $where = '' ][, string $orderBy = '' ][, int $limit = 0 ][, int $offset = 0 ]) : array<string|int, mixed>

Cette fonction construit et exécute une requête SELECT sur la table spécifiée. Elle permet d'appliquer une clause WHERE pour filtrer les enregistrements, une clause ORDER BY pour trier les résultats, ainsi qu'une pagination via les paramètres limit et offset. Noter qu'une valeur de 0 pour le paramètre limit désactive la limitation du nombre de lignes retournées.

Parameters
$table : string

Nom de la table cible.

$where : string = ''

(Optionnel) Clause WHERE pour filtrer les enregistrements.

$orderBy : string = ''

(Optionnel) Clause ORDER BY pour trier les résultats.

$limit : int = 0

(Optionnel) Nombre maximum de lignes à récupérer. Une valeur de 0 désactive la limite.

$offset : int = 0

(Optionnel) Position de départ pour la récupération des enregistrements.

Return values
array<string|int, mixed> —

Tableau contenant l'ensemble des enregistrements récupérés.

fetchOne(string $table, string $where[, string $orderBy = '' ][, array<string|int, mixed> $params = [] ]) : array<string|int, mixed>|false

$params : array<string|int, mixed> = []

array<string|int, mixed>|false —

Tableau associatif représentant l'enregistrement trouvé ou false si aucun enregistrement ne correspond.

updateRow()

db.php : 198

Met à jour les enregistrements d'une table avec les données fournies.

updateRow(string $table, array<string|int, mixed> $data, string $where[, array<string|int, mixed> $whereParams = [] ]) : int

Cette fonction valide le nom de la table, construit dynamiquement la clause SET en encapsulant les noms de colonnes avec des backticks et en utilisant des placeholders préfixés par "set_", puis exécute une requête SQL UPDATE avec une clause WHERE personnalisée. Si le tableau de données est vide, une exception est levée. En cas d'erreur lors de l'exécution de la requête, l'erreur est loguée et une exception PDO est relancée. Utilise des placeholders nommés pour les clauses SET et WHERE.

Parameters
$table : string

Nom de la table cible.

$data : array<string|int, mixed>

Tableau associatif contenant les colonnes et leurs nouvelles valeurs (pour SET).

$where : string

Clause SQL WHERE avec des placeholders nommés (ex: "id = :where_id").

$whereParams : array<string|int, mixed> = []

Tableau associatif des valeurs pour les placeholders de la clause WHERE (ex: [':where_id' => 1]).

Tags
throws
Exception

Si le tableau $data est vide.

throws
PDOException

Si une erreur survient lors de l'exécution de la requête.

Return values
int —

Nombre d'enregistrements mis à jour.
