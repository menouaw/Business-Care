paginateResults()

functions.php : 283

Récupère et pagine les résultats d'une requête sur une table.

paginateResults(string $table, int $page[, int $perPage = DEFAULT_ITEMS_PER_PAGE ][, string $where = '' ][, string $orderBy = '' ]) : array<string|int, mixed>

Cette fonction calcule le nombre total d'enregistrements correspondant aux critères optionnels, détermine le nombre total de pages, ajuste le numéro de page pour qu'il soit valide, et récupère ensuite les enregistrements de la page courante en appliquant les clauses facultatives de filtrage et de tri.

Parameters
$table : string

Nom de la table à interroger.

$page : int

Numéro de la page courante.

$perPage : int = DEFAULT_ITEMS_PER_PAGE

Nombre d'éléments à afficher par page.

$where : string = ''

(Facultatif) Clause WHERE pour filtrer les résultats.

$orderBy : string = ''

(Facultatif) Clause ORDER BY pour trier les résultats.

Return values
array<string|int, mixed> —

Un tableau associatif contenant :

  • 'items' (array) : Les enregistrements de la page courante.
  • 'currentPage' (int) : Le numéro de page ajusté.
  • 'totalPages' (int) : Le nombre total de pages.
  • 'totalItems' (int) : Le nombre total d'enregistrements répondant aux critères.
  • 'perPage' (int) : Le nombre d'éléments par page.

renderPagination()

functions.php : 328

Génère et renvoie une interface de pagination au format Bootstrap.

renderPagination(array<string|int, mixed> $pagination, string $urlPattern) : string

Cette fonction crée le code HTML d'une pagination basée sur les informations contenues dans le tableau $pagination. Si le nombre total de pages est inférieur ou égal à 1, elle retourne une chaîne vide.

Parameters
$pagination : array<string|int, mixed>

Tableau associatif issu de paginateResults() contenant les clés 'currentPage' et 'totalPages'.

$urlPattern : string

Motif d'URL incluant le placeholder "{page}" qui sera remplacé par le numéro de page.

Return values
string —

Le code HTML de l'interface de pagination ou une chaîne vide si aucune pagination n'est nécessaire.
