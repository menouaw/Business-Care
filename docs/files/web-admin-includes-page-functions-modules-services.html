servicesSave()

services.php : 107

Crée ou met à jour un service dans la base de données.

servicesSave(array<string|int, mixed> $data[, int $id = 0 ]) : array<string|int, mixed>

Cette fonction vérifie que les données obligatoires (nom, prix et type) sont présentes et valides. En cas d'erreur de validation, elle retourne un tableau contenant les messages d'erreur. Si les validations réussissent, elle effectue une mise à jour si un identifiant supérieur à 0 est fourni, ou crée un nouveau service sinon. En cas d'échec de l'opération en base de données, un tableau d'erreurs est retourné.

Parameters
$data : array<string|int, mixed>

Les informations du service, incluant notamment 'nom', 'prix', 'type', et d'autres champs optionnels.

$id : int = 0

L'identifiant du service à mettre à jour, ou 0 pour créer un nouveau service.

Return values
array<string|int, mixed> —

Tableau associatif indiquant le succès de l'opération. En cas de succès, il contient une clé 'message'; en cas d'erreur, une clé 'errors' avec la liste des messages d'erreur.

servicesDelete()

services.php : 216

Supprime un service après vérification des dépendances associées.

servicesDelete(int $id) : array<string|int, mixed>

Cette fonction tente de supprimer un service identifié par son ID, après avoir vérifié qu'il n'a aucun rendez-vous ou évaluation associé. Si des dépendances sont présentes, la suppression est annulée et une opération d'échec est consignée. Sinon, le service est supprimé et l'opération de suppression est enregistrée.

Parameters
$id : int

L'identifiant du service à supprimer.

Return values
array<string|int, mixed> —

Un tableau associatif contenant un booléen sous la clé 'success' indiquant si l'opération a réussi et un message descriptif sous la clé 'message'.
